# 堆

堆（Heap）是一种特殊的 **树形数据结构**，通常指 **二叉堆**（Binary Heap）。它是一棵 **完全二叉树**，并且满足一个关键特性：

- 大顶堆（Max Heap）：任意节点的值 ≥ 子节点的值；
- 小顶堆（Min Heap）：任意节点的值 ≤ 子节点的值；

换句话说，堆保证了 **堆顶元素永远是最大值或最小值**。

>可以把它想象成“层级严格、位置紧凑的树”，但重点是：堆不是用来查找任意元素的，而是用来快速得到最大/最小值的结构。

> [!NOTE] 堆的优势
>
> - **获取最值非常快**：最大值/最小值用于在堆顶，直接取即可，因此时间复杂度为 $O(1)$；
> - **插入和删除很快**：插入时，把元素加到最后，然后上浮，删除时把最后一个元素提到堆顶，再下沉，因此时间复杂度为 $O(logn)$；
> - **占用空间紧凑**：二叉堆通常用数组实现，不需要指针，内存连续、存储效率高；
